# Insurance Management API

A backend REST API for managing insurance policies, clients, and claims.

## Features

- Client management
- Insurance policy management
- Claims creation and tracking
- Validation and business rules
- Database persistence
- RESTful endpoints

## Tech Stack

- Java / Spring Boot
- PostgreSQL
- Maven

## API

The API exposes endpoints for:
- clients
- policies
- claims
